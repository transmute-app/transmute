# Transmute Frontend

React + TypeScript + Vite + Tailwind CSS frontend for Transmute file conversion service.

## Development

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

## Build

Build for production:
```bash
npm run build
```

This will generate static files in the `dist/` directory that can be served by FastAPI.

## Preview Production Build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
